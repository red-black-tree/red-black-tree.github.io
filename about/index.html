关于博客

这个网站是本人博客的 main branch，另有 luogu branch 在此。

其中 main branch 整活博客更多一点，并将部分题解与 luogu branch 同步。

而 luogu branch 仅用于发布题解与重要公告。然而不支持手动更改创建时间。

预计今后将重心放在 main branch，不保证 luogu branch 一直维护[1]。

另外特别鸣谢 wdsean 和 cmk666 对 main branch 的技术支持。

关于日期

日期都是虚构的，根据 date[2] 改变置顶等级。

  • 1582-10-14：公告，time 根据置顶程度编码。
  • 1582-10-07：整活，time 根据创建时间编码。
  • 1582-10-06：题解，time 根据创建时间编码。
  • 1582-10-05：测试，time 根据创建时间编码。

然而 time 在编译前后总有 43 秒的偏差，原因不明，必须手动调整。

此外，若有紧急公告，则会将日期设为 9999-99-99，并在紧急状态结束后调至 1582-10-13 永久保存。
